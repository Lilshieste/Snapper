How to use Snapper
Using Snapper to capture images from web pages is simple:
  1. Navigate to the web page containing the information you want to snapshot.
  2. Enable Snapper by using one of these methods:
    • Click the Snapper icon, on the bottom status panel of the window.
    • Bring up a context-menu on the page, and select "Start Snapping"
    • Bring up a context-menu on the Snapper icon, and select "Start Snapping"
  3. Click and drag on the webpage to select the area to be included in the snapshot.
  4. After the area selection has been made, a dialog will appear to save the snapshot.
Note: After successfully taking a snapshot, Snapper automatically resets to allow normal navigation on the web page.
Using the Different Snapping Modes
Starting with release 2.0, Snapper allows users to snap their images in multiple ways. The Snapping Mode may be specified in Snapper's Options window, by right-clicking on the Snapper icon and selecting "Options...".
  • Simple Snapping - This is the default behavior of Snapper. To snap an image, the user enables Snapper, clicks the mouse button, and designates the area to be snapped. Immediately after the selection is made, Snapper copies and/or saves the snapped image, and allows the user to resume normal webpage browsing.
  • Anchored Snapping - A new mode introduced in version 2.0. At first, this mode behaves in a similar manner to Simple Snapping. That is, the user enables Snapper, and designates an area on the web page to snap. After doing so, though, Snapper allows the user to move and resize the snapping area, for finer control over the snapped image. After adjusting the area to be snapped, the user then double-clicks on the snapping area to complete the snapping.

Copying Snapped Images to the Clipboard
Copying snapped images to your system's clipboard is just as easy as saving them to a file. Navigate to Snapper's options page (right-click on the Snapper icon, and select "Options..."), where you still see a collection of radio button options:
  • Copy image to clipboard - Only copy the snapped image to the clipboard (does not save)
  • Save image to file - Only save the snapped image to a file (does not copy)
  • Copy & Save - Copies the snapped image to the clipboard *and* saved it to a file
