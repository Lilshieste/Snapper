Extensão Snapper versão 2.1
Resumo
Snapper foi criado para ser uma ferramenta que pode facilmente salvar imagens capturadas de áreas específicas de páginas web. Isto ajuda a aliviar a necessidade de cortar capturas de tela só para remover informações irrelevantes.
