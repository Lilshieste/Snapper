Extensión Snapper versión 2.1
Resumen
Snapper fué creada para ser una herramienta que permita guardar facilmente imágenes instantáneas de áreas específicas de las páginas web Esto nos ayuda aliviandonos de la necesidad de tener que editar la Capturas de Pantalla (screenshots) tan sólo para recortar ciertas zonas de la imagen y así remover información irrelevante, innecesaria o sin importancia.

Notas Especiales - Por favor lealas atentamente
Hay que recalcar ciertas cosas sobre el actual comportamiento de Snapper:
  • Para que funcione correctamente, Snapper requiere de las Java Runtime Environment (JRE) versión 1.4 o posterior.
  • En algunas ocaciones, Snapper es incapaz de tomarle una instantánea a alguna película incrustada Shockwave/Flash mostrada en la página web.
